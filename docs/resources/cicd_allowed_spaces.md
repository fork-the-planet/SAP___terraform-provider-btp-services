# btpservice_cicd_allowed_spaces (Resource)

Manages the list of allowed spaces in the SAP BTP CI/CD service.

This is a **bulk-replace** resource: the entire list is replaced on every apply. Spaces not listed will lose access to CI/CD service instances.

~> **Warning:** Only one `btpservice_cicd_allowed_spaces` resource must exist per provider configuration. Defining multiple instances will cause them to overwrite each other on every apply, leading to unpredictable state and loss of access for spaces managed by the other instance.

## Example Usage

```terraform
resource "btpservice_cicd_allowed_spaces" "this" {
  allowed_spaces = [
    {
      space_guid = "a2bcf2b8-6eda-5b8a-0b7c-8512bb82060f"
      comment    = "Team Alpha production space"
    },
    {
      space_guid = "d794d687-3053-4cba-a942-88e6b13ef035"
      comment    = "Team Beta staging space"
    },
  ]
}
```

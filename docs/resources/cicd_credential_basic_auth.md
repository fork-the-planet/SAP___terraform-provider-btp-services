# btpservice_cicd_credential_basic_auth (Resource)

Manages a Basic Authentication credential in the SAP BTP CI/CD service.

## Example Usage

```terraform
resource "btpservice_cicd_credential_basic_auth" "example" {
  name        = "my-deploy-user-from-cicd"
  description = "CF deployment user1"
  username    = "deploy-user@example.com"
  password    = "passoword"
}
```

# btpservice_cicd_credential_container_registry (Resource)

Manages a Container Registry configuration credential in the SAP BTP CI/CD service.

## Example Usage

```terraform
resource "btpservice_cicd_credential_container_registry" "example" {
  name        = "my-container-registry"
  description = "Container registry credentials for Docker Hub"
  content = jsonencode({
    auths = {
      "registry.example.com" = {
        auth = base64encode("username:password")
      }
    }
  })
}
```
